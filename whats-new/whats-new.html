August 3rd, 2025

Advanced Settings Rework

The "Action & Player Timing" panel has been updated with the following key features for a cleaner, more intuitive experience:

  • New Collapsible Panel: For a cleaner interface, advanced controls are now located in a "More settings" section. This includes:
    • Action Delay: A small delay before seeking to prevent stuttering.
    • Seek Throttle: Prevents spamming rapid skip commands.
    • Seek Interval: Controls the speed of continuous seeking when a hotkey is held down.
    • Controls Visible Duration: Sets how long player controls stay visible after a skip.
  • Buffering Protection Toggle: Added a new toggle to control the buffering protection feature, allowing you to enable or disable protection against YouTube's problematic black screen buffering state. Disabling will allow you to seek through even when the player is buffering.

Understanding the Advanced Settings

Action Delay
Adds a small delay between when you press a hotkey and when the seek happens. Helps prevent stuttering on some systems by giving YouTube's player time to handle the seek request. Increase if you notice video stuttering when skipping, decrease for more immediate seeking on fast systems.
Default: 20ms
Seek Throttle
Minimum time between separate skip commands to prevent overwhelming YouTube's player. Lower values allow faster key presses to register, higher values provide more stability on slower systems or connections.
Default: 100ms
Seek Interval
Controls how fast continuous seeking happens when you hold down a hotkey. Lower values = faster scanning through videos, higher values = more precise, controlled seeking.
Default: 150ms
Controls Visible Duration
How long YouTube's player controls stay visible after you skip. Ensures you can see the progress bar and time display after seeking. Set higher to see controls longer, or lower for a cleaner viewing experience.
Default: 2500ms
Buffering Protection
Prevents seeks that might trigger YouTube's black screen buffering state, which can cause the player to lock up or restart the video. This protection was always active before - now you can choose to disable it if you prefer unrestricted seeking.
Default: Enabled

June 21st, 2025

This is a major release focused on giving you more control over your viewing experience, redesigning the UI for clarity, and improving overall performance.

New Features & Major Enhancements

  • Fully Customizable Hotkeys: Take complete control of your keyboard. You can now assign custom keys for skipping forward and backward (e.g., J and L). The UI prevents you from using the same key for both actions. A new reset button lets you restore the default hotkeys (← and →) instantly.
  • New Advanced Settings Page: A dedicated section for power users has been added. Action Delay Control: Fine-tune player responsiveness with a new experimental setting to add a small delay (in ms) after a skip command. Perfect for preventing buffering on some systems.

UI & UX Improvements

  • Collapsible Setting Panels: Preset and hotkey editors are now neatly tucked away in collapsible panels. Click the edit icon to expand and customize, keeping the interface clean and focused.
  • Clear Visual States: Disabled sections are now grayed out and their editors are automatically collapsed, providing a clear visual cue for which settings are active.
  • Improved Number Inputs: All number fields have been upgraded to custom spinner controls with up and down buttons for quick and easy value adjustments.
  • Enhanced User Guidance: New tooltips have been added to explain advanced features like Action Delay, ensuring you know exactly what each setting does.

Under-the-Hood Fixes & Improvements

  • Robust Settings Migration: Your settings are now automatically and safely preserved across future extension updates. No more resets.
  • Smoother Button Animations: The on-screen forward and rewind buttons now appear with a much smoother, flicker-free animation.
